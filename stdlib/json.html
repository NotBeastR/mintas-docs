JSON Module

The json module provides powerful parsing and serialization for JSON data.

Importing

include json

Encoding & Decoding

Function Description Example
encode(val) Converts a Mintas value to a JSON string. json.encode({a: 1})
decode(str) Parses a JSON string into a Mintas value. let data = json.decode("{\"a\": 1}")
pretty(val) Converts a value to a formatted JSON string. print(json.pretty(data))
is_valid(str) Returns true if the string is valid JSON. if json.is_valid(input) { ... }

Manipulation

Work with JSON strings directly without full decoding.

Function Description Example
get(str, path) extracts a value using dot-notation. json.get(json_str, "users.0.name")
set(str, key, val) Sets a value in the JSON string. let new_json = json.set(old_json, "status", "active")
keys(str) Returns keys if the root is an object. let keys = json.keys(json_str)

Aliases

  • stringify -> encode
  • parse -> decode

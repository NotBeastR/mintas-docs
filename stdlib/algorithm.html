Algorithm Module

Common computer science algorithms and data structure operations.

Sorting

include algorithm

numbers = [3, 1, 4, 1, 5, 9, 2, 6]
sorted = algorithm.sort(numbers)
say(sorted)  # [1, 1, 2, 3, 4, 5, 6, 9]

# Reverse sort
desc = algorithm.sort(numbers, "desc")
say(desc)  # [9, 6, 5, 4, 3, 2, 1, 1]

Array Operations

include algorithm

# Reverse
reversed = algorithm.reverse([1, 2, 3, 4])
say(reversed)  # [4, 3, 2, 1]

# Unique values
unique = algorithm.unique([1, 2, 2, 3, 3, 3])
say(unique)  # [1, 2, 3]

# Shuffle
shuffled = algorithm.shuffle([1, 2, 3, 4, 5])

Search

include algorithm

arr = [1, 2, 3, 4, 5, 6, 7, 8, 9]

# Binary search (requires sorted array)
index = algorithm.binary_search(arr, 5)
say(index)  # 4

# Linear search
found = algorithm.find(arr, 7)
say(found)  # 6

API Reference

  • algorithm.sort(array, order?) - Sort array
  • algorithm.reverse(array) - Reverse array
  • algorithm.unique(array) - Remove duplicates
  • algorithm.shuffle(array) - Randomize order
  • algorithm.binary_search(array, value) - Binary search
  • algorithm.find(array, value) - Linear search

Dew Web Framework - Complete API Reference

The dew module is a comprehensive, production-ready web framework for Mintas. It includes routing, middleware, WebRTC, JavaScript integration, databases, and advanced styling for modern web development.

Quick Start

include dew

let app = dew.create_server({port: 3000})

@app.get("/"):
    return dew.text("Hello from Mintas!")
end

dew.serve(app)

Server Lifecycle

Function Description Example
create_server(options) Creates a new server configuration. Options: port, host, debug. let app = dew.create_server({port: 3000})
serve(server) Starts the server and blocks. Listening on configured port. dew.serve(app)

Routing

Routes map HTTP methods and paths to handler functions.

Function Description Example
get(path, handler) Registers a GET route. @app.get("/") ... end
post(path, handler) Registers a POST route. @app.post("/submit") ... end
put(path, handler) Registers a PUT route. @app.put("/update") ... end
delete(path, handler) Registers a DELETE route. @app.delete("/remove") ... end
static(url, dir) Serves static files from a directory. app.static("/assets", "./public")
use(handler) Registers middleware that runs for every request. app.use(middleware)

Request Object (getback)

The handler function receives a request object (often called `req` or `getback`).

  • req.body - Raw request body as string.
  • req.json() - Parsed JSON body as a table.
  • req.query(key) - Get URL query parameter value.
  • req.param(key) - Get dynamic path parameter.
  • req.header(key) - Get request header value.
  • req.method - HTTP method: GET, POST, PUT, DELETE, etc.
  • req.path - The requested route path.
  • req.ip - Client's IP address.

Database & Authentication

Function Description Example

WebRTC for Real-Time Communication NEW

Build peer-to-peer applications with low-latency connections. Perfect for video calls, gaming, collaborative tools, and live streaming.

Core WebRTC Functions

Function Purpose Example
webrtc_peer(config) Creates a new WebRTC peer connection let peer = dew.webrtc_peer({id: "user1"})
webrtc_offer(peer) Generates an SDP offer for connection initiation let offer = dew.webrtc_offer(peer)
webrtc_answer(offer) Creates an SDP answer to a received offer let answer = dew.webrtc_answer(offer)
webrtc_datachannel(peer, config) Creates a bidirectional data channel let dc = dew.webrtc_datachannel(peer, {label: "chat"})
webrtc_send(channel, data) Sends data through a data channel dew.webrtc_send(dc, "Hello!")
webrtc_on_message(channel, handler) Registers a message handler dew.webrtc_on_message(dc, handler)
webrtc_close(peer_or_channel) Closes a peer connection or data channel dew.webrtc_close(peer)
webrtc_stats(peer) Gets real-time connection statistics let stats = dew.webrtc_stats(peer)

See the complete WebRTC guide for real-world examples and best practices.

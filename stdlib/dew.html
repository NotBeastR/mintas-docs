Dew Web Framework

The dew module is a powerful, integrated web server and framework for Mintas. It supports routing, middleware, templates, static file serving, and databases.

Importing

include dew

Server Lifecycle

Function Description Example
create_server(options) Creates a new server configuration. Options: port, host, debug. let app = dew.create_server({port: 3000, debug: true})
serve(server) Starts the server. This function blocks. dew.serve(app)

Routing

Routes map HTTP methods and paths to handler functions.

Function Description Example
get(path, handler) Registers a GET route. app.get("/", fn(req) { return dew.text("Hello") })
post(path, handler) Registers a POST route. app.post("/submit", fn(req) { ... })
static(url, dir) Serves static files from a directory. app.static("/assets", "./public")
use(handler) Registers middleware that runs for every request. app.use(logger_middleware)

Request Object

The handler function receives a request object (often called `req` or `getback`).

  • req.body - Raw body string.
  • req.json - Parsed JSON object (if content-type is json).
  • req.query - Table of query parameters.
  • req.params - Table of URL parameters (e.g. /user/:id).
  • req.headers - Table of request headers.

Response Helpers

Function Description Example
text(str, status) Returns a plain text response. return dew.text("OK", 200)
json(obj, status) Returns a JSON response. return dew.json({msg: "Hi"}, 200)
html(str, status) Returns an HTML response. return dew.html("

Hi

", 200)
file(path) Returns a file download. return dew.file("./report.pdf")
redirect(url) Redirects to another URL. return dew.redirect("/login")

Database & Magic

Function Description Example
query(config, sql, params) Executes a SQL query (SQLite/Postgres). Returns array of rows. dew.query(db_conf, "SELECT * FROM users", [])
hash_password(pwd) Securely hashes a password using Bcrypt. let h = dew.hash_password("secret")
verify_password(pwd, hash) Verifies a password against a hash. if dew.verify_password("secret", h) { ... }
uuid() Generates a random UUID v4 string. let id = dew.uuid()

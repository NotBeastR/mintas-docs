Standard Library

Mintas comes with a "Batteries Included" philosophy, offering a vast array of built-in modules for everything from file manipulation to web requests and database interaction.

Included Modules

Core

  • math: Constants and advanced mathematical functions.
  • sysfiles: File system operations.
  • datetime: Handling dates and times.
  • system: OS, Env, Path, and Processes.

Web & Game

  • dew: Full-stack Web Framework.
  • canvas: 2D Game Engine & Graphics.
  • networking: HTTP & WebSocket clients.

Protocols

  • dns: Domain resolution.
  • ftp: File Transfer Protocol.
  • mqtt: IoT messaging.
  • smtp: Sending emails.

Data & Serialization

  • json: JSON manipulation.
  • database: SQLite and Redis support.
  • formats: CSV, YAML, PDF, QR Codes.

AI & Security

  • openai: Integrated AI support.
  • security: Crypto, Hash, UUID, Base64.

Utilities

  • utilities: CLI, Colors, Validation, Cache.
  • algorithms: Common CS algorithms.

Using Modules

Use the include keyword to bring a module into your script.

include math
include sysfiles as fs

say(math.PI)
fs.write("data.txt", "Hello!")

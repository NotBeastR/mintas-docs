AI Integration

Modules for interacting with AI services and large language models.

OpenAI Module

Client for OpenAI API services, including Chat, Completion, and Image Generation.

include openai
[!NOTE] Requires the networking feature to be enabled.
Function Description Example
chat_completion(key, msgs, [model]) Generates a chat response. openai.chat_completion(key, [{role: "user", content: "Hi"}])
completion(key, prompt, [model]) Generates a text completion. openai.completion(key, "Once upon a time")
image_generation(key, prompt, [size]) Generates images via DALL-E. openai.image_generation(key, "A red cat")
embedding(key, input, [model]) Generates vector embeddings for a string. openai.embedding(key, "Search query")
moderation(key, input) Checks if text violates content policies. openai.moderation(key, "Some text")

Configuration

Function Description Example
set_base_url(url) Overrides the API base URL (defaults to api.openai.com). openai.set_base_url("https://my-proxy.com")

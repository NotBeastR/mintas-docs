Protocols

Support for various network protocols including DNS, FTP, MQTT, and SMTP.

DNS Module

Domain Name System lookups.

include dns
Function Description Example
lookup(host) Resolves a hostname to IP addresses. dns.lookup("google.com")
mx(domain) Look up MX records for a domain. dns.mx("gmail.com")

FTP Module

File Transfer Protocol client.

include ftp
Function Description Example
connect(cfg) Connects to an FTP server. let conn = ftp.connect({host: "..."})
list(conn, [path]) Lists files in a directory. ftp.list(conn, "/")
download(conn, remote, local) Downloads a file. ftp.download(conn, "r.txt", "l.txt")

MQTT Module

Message Queuing Telemetry Transport client.

include mqtt
Function Description Example
connect(host, [port]) Connects to an MQTT broker. let client = mqtt.connect("localhost")
publish(client, topic, msg) Publishes a message. mqtt.publish(client, "home/temp", "22")

SMTP Module

Simple Mail Transfer Protocol client.

include smtp
Function Description Example
send(cfg) Sends an email. smtp.send({to: "...", body: "..."})

Ping & Traceroute

Network connectivity utilities.

include ping
Function Description Example
host(host) Pings a host and returns stats. ping.host("8.8.8.8")
port(host, port) Checks if a port is open. ping.port("localhost", 80)
trace(host) Performs a traceroute. ping.trace("google.com")

Webhook Module

Send and verify webhooks.

include webhook
Function Description Example
send(url, payload) Sends a webhook payload. webhook.send(url, {event: "ping"})
verify(body, sig, secret) Verifies a webhook signature. webhook.verify(body, sig, secret)

GraphQL Module

GraphQL client and query builder.

include graphql
Function Description Example
query(url, query, [vars]) Sends a GraphQL query. graphql.query(url, "{ user { name } }")
build_query(op, name, fields) Builds a GraphQL query string. graphql.build_query("query", "GetUser", ["id", "name"])

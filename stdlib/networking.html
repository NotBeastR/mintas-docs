Networking Modules

Mintas provides modules for making HTTP requests and connecting to WebSockets.

HTTP Requests

The requests module allows you to act as an HTTP client.

include requests
Function Description Example
get(url) Performs a GET request. Returns a table with `status` and `text`. let res = requests.get("https://api.example.com")
post(url, json) Performs a POST request with optional JSON body. requests.post("https://api.example.com/login", {u: "me"})

WebSocket Client

The sockets module handles WebSocket connections.

include sockets
Function Description Example
connect(url) Connects to a WebSocket server. sockets.connect("wss://echo.websocket.org")

Note: to build a Web Server, use the Dew Framework instead.

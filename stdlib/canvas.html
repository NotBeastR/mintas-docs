Canvas Module

The canvas module is the Ultimate 2D Game Framework for Mintas. It provides an easy-to-use interface for creating games with sprites, physics, collision detection, and audio.

Importing

include canvas

Window Management

Function Description Example
create(title, width, height) Opens a new game window. canvas.create("My Game", 800, 600)
update() Updates the window and inputs. Call this in your game loop. Returns false if window is closed. while canvas.update() { ... }
clear(color) Clears the screen with a hex color. canvas.clear("#000000")
quit() Closes the game window. canvas.quit()

Drawing Primitives

Function Description Example
rect(x, y, w, h, color) Draws a rectangle outline. canvas.rect(10, 10, 50, 50, "#FF0000")
fill_rect(x, y, w, h, color) Draws a filled rectangle. canvas.fill_rect(10, 10, 50, 50, "#00FF00")
circle(cx, cy, r, color) Draws a circle outline. canvas.circle(100, 100, 30, "#0000FF")
fill_circle(cx, cy, r, color) Draws a filled circle. canvas.fill_circle(100, 100, 30, "#FFFF00")
line(x1, y1, x2, y2, color) Draws a line between two points. canvas.line(0, 0, 800, 600, "#FFFFFF")
text(str, x, y, color, scale) Draws text on screen. Scale defaults to 2. canvas.text("Score: 100", 10, 10, "#FFFFFF", 2)

Sprite System

Sprites are the core objects in Canvas. They handle position, velocity, physics, and rendering automatically.

Function Description Example
sprite(id, x, y, w, h, color) Creates a new sprite. canvas.sprite("player", 100, 100, 32, 32, "#00FF00")
set(id, prop, value) Sets a property of a sprite (x, y, vx, vy, gravity, etc.). canvas.set("player", "gravity", 0.5)
get(id, prop) Gets a property of a sprite. let x = canvas.get("player", "x")
draw_all() Draws all visible sprites. canvas.draw_all()
set_texture(id, tex_name) Assigns a loaded image texture to a sprite. canvas.set_texture("player", "hero_idle")
set_frame(id, x, y, w, h) Sets the animation frame (sub-rectangle) of the texture. canvas.set_frame("player", 0, 0, 32, 32)

Assets & Audio

Function Description Example
load_image(name, path) Loads an image file for use as a texture. canvas.load_image("hero", "assets/hero.png")
play_sound(path) Plays a sound file (WAV/MP3/OGG). Fire and forget. canvas.play_sound("assets/jump.wav")

Input

Function Description Example
key(name) Returns true if key is currently held down. (left, right, up, down, space, a-z...) if canvas.key("right") { ... }
key_pressed(name) Returns true if key was pressed this frame (one-shot). if canvas.key_pressed("space") { ... }
mouse_down(button) Returns true if mouse button is held (0=left, 1=right). if canvas.mouse_down(0) { ... }

Database Modules

Mintas provides multiple ways to interact with databases. You can use the high-level dew.query for SQL databases (recommended for web apps) or specific modules for low-level control.

Universal Query (via Dew)

Access SQLite, Postgres, or MySQL using a unified API.

include dew
Function Description Example
dew.database(str) Parses a connection string into a config object. let db = dew.database("sqlite:///app.db")
dew.query(db, sql, params) Executes raw SQL. Returns an array of tables. let rows = dew.query(db, "SELECT * FROM x WHERE id = ?", [1])
dew.redis_get(url, key) Quickly fetch a string from Redis. let val = dew.redis_get("redis://localhost/", "key")

SQLite3 Module

A dedicated module for SQLite operations.

include sqlite3
Function Description Example
connect(path) Opens a connection to an SQLite file. let conn = sqlite3.connect("data.db")
create_table(name, cols) Creates a table with specified columns. sqlite3.create_table("users", {id: "INTEGER", name: "TEXT"})
insert(table, data) Inserts a row into a table. sqlite3.insert("users", {name: "Bob", age: 30})
select(table) Selects all rows from a table. let users = sqlite3.select("users")
delete(table, where) Deletes rows matching conditions. sqlite3.delete("users", {name: "Bob"})

Redis Module

A dedicated module for Redis operations including Lists and Hashes.

include redis2
Function Description Example
connect(host) Connects to a Redis server. Default: "localhost:6379". redis2.connect()
set(key, value) Sets a string key. redis2.set("session", "active")
get(key) Gets a string key. let val = redis2.get("session")
push(list, value) Pushes a value to a list (LPUSH). redis2.push("queue", "job1")
pop(list) Pops a value from a list (LPOP). let job = redis2.pop("queue")
hash_set(key, field, val) Sets a field in a hash. redis2.hash_set("user:1", "name", "Bob")
hash_get_all(key) Gets all fields from a hash. let user = redis2.hash_get_all("user:1")

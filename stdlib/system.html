System & OS

Modules for interacting with the operating system, file paths, and external processes.

OS Module

include os
Function Description Example
platform() Returns OS name (e.g., "linux", "windows"). os.platform()
cwd() Returns current working directory. os.cwd()
chdir(path) Changes current working directory. os.chdir("/tmp")
exit(code) Exits the process with the given code. os.exit(1)
homedir() Returns user's home directory. os.homedir()

Env Module

include env
Function Description Example
get(key, [def]) Gets env var, optionally with default. env.get("PATH")
set(key, val) Sets environment variable. env.set("FOO", "bar")
load(path) Loads variables from a .env file. env.load(".env")

Path Module

include path
Function Description Example
join(...) Joins path segments intelligently. path.join("a", "b", "c")
basename(p) Returns the file name portion. path.basename("/a/b.txt")
extname(p) Returns the extension. path.extname("file.txt")
exists(p) Checks if path exists. path.exists("file.txt")

Subprocess Module

include subprocess
Function Description Example
run(cmd) Runs command, returns object with stdout/stderr. let res = subprocess.run("ls -la")
shell(cmd) Runs command purely via shell (cmd/sh). subprocess.shell("echo 'hello'")
spawn(cmd) Spawns process in background. subprocess.spawn("notepad")

Buffer Module

Low-level binary data handling.

include buffer
Function Description Example
create(size) Allocates a new buffer. let b = buffer.create(1024)
from(str) Creates buffer from string. let b = buffer.from("hello")
to_hex(buf) Converts buffer to hex string. buffer.to_hex(b)

Timer Module

include timer
Function Description Example
sleep(ms) Pauses execution for milliseconds. timer.sleep(1000)
start(name) Starts a named timer. timer.start("bench")
stop(name) Stops timer and returns elapsed time stats. let stats = timer.stop("bench")

See Also: SysFiles Module for file reading/writing operations.

Math Module

The math module provides essential mathematical constants and functions for advanced calculations.

Importing

include math

Constants

Constant Value
math.pi 3.141592653589793
math.e 2.718281828459045

Functions

Function Description Example
abs(x) Absolute value of x. math.abs(-10) # 10
sqrt(x) Square root of x. math.sqrt(16) # 4
pow(x, y) x raised to the power of y. math.pow(2, 3) # 8
sin(x) Sine of x (in radians). math.sin(0) # 0
cos(x) Cosine of x (in radians). math.cos(0) # 1
tan(x) Tangent of x (in radians). math.tan(0) # 0
log(x) Natural logarithm of x. math.log(1) # 0
log10(x) Base-10 logarithm of x. math.log10(100) # 2
ceil(x) Smallest integer greater than or equal to x. math.ceil(4.2) # 5
floor(x) Largest integer less than or equal to x. math.floor(4.8) # 4
random() Random number between 0 and 1. math.random()

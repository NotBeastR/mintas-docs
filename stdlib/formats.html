Data Formats

Modules for parsing, generating, and manipulating various data formats like CSV, YAML, PDF, and QR codes.

CSV Module

Read and write Comma Separated Values.

include csv
Function Description Example
read(path, [has_header]) Reads a CSV file into an array of tables or arrays. let data = csv.read("users.csv")
parse(content, [has_header]) Parses a CSV string. csv.parse("name,age\nAlice,30")
write(path, data) Writes an array of tables/arrays to a CSV file. csv.write("out.csv", data)

YAML Module

Parsing and generating YAML data structures.

include myyaml as yaml
Function Description Example
parse(str) Parses a YAML string into a Mintas table. let cfg = yaml.parse(content)
stringify(val) Converts a table/array to a YAML string. yaml.stringify({name: "App"})

PDF Module

Generate PDF documents programmatically.

include mypdf as pdf
Function Description Example
create(title) Creates a new PDF document object. let doc = pdf.create("Invoice")
add_page(doc) Adds a new page to the document. doc = pdf.add_page(doc)
add_text(doc, text, x, y) Adds text at specific coordinates. doc = pdf.add_text(doc, "Hello", 50, 700)
save(doc, path) Saves the PDF to a file. pdf.save(doc, "output.pdf")

QR Code Module

Generate QR codes in various formats.

include myqr as qr
Function Description Example
to_ascii(data) Generates a QR code as an ASCII string. print(qr.to_ascii("https://mintas.dev"))
to_svg(data) Generates an SVG string of the QR code. let svg = qr.to_svg("data")
to_html(data) Generates an HTML representation. res.body = qr.to_html("data")

Archive & Compression

Work with TAR archives and compression algorithms.

include archive
include compress
Function Description Example
archive.create(name) Creates a new archive object. let a = archive.create("logs.tar")
compress.gzip(data) Compresses data using GZIP. let c = compress.gzip(content)
compress.gunzip(data) Decompresses GZIP data. compress.gunzip(c)

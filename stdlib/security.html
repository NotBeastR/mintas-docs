Security & Cryptography

Modules for hashing, encryption, secure connections, and encoding.

Crypto Module

include crypto
Function Description Example
encrypt(data, key) Encrypts string using a simple XOR cipher. let enc = crypto.encrypt("secret", "key")
decrypt(data, key) Decrypts XOR-encrypted string. let dec = crypto.decrypt(enc, "key")
hash(data, algo) Hashes data. Algo: "md5", "sha1", "sha256". crypto.hash("foo", "sha256")
hmac(data, key) Computes HMAC-SHA256. crypto.hmac("data", "secret")
random_bytes(n) Returns array of n random bytes. crypto.random_bytes(16)
random_hex(n) Returns random hex string of length n/2 bytes. crypto.random_hex(32)

Hash Module

Dedicated hashing algorithms.

include hash
Function Description Example
sha256(str) Returns SHA-256 hash. hash.sha256("hello")
sha512(str) Returns SHA-512 hash. hash.sha512("hello")
md5(str) Returns MD5 hash. hash.md5("hello")
crc32(str) Returns CRC32 checksum. hash.crc32("hello")

UUID Module

include uuid
Function Description Example
v4() Generates a random UUID (v4). uuid.v4()
v1() Generates a time-based UUID (v1). uuid.v1()
is_valid(str) Checks if string is a valid UUID. uuid.is_valid(id)

Base64 Module

include base64
Function Description Example
encode(str) Encodes string to Base64. base64.encode("hello")
decode(str) Decodes Base64 string. base64.decode("AGVsbG8=")
urlsafe_encode(str) URL-safe Base64 encoding. base64.urlsafe_encode("hello")

SSH Module

include ssh
Function Description Example
connect(config) Connects to SSH server. Config: {host, user, pass, key}. let conn = ssh.connect({host: "10.0.0.1", user: "root"})
exec(conn, cmd) Executes command on remote server. Returns table with stdout/stderr. ssh.exec(conn, "ls -la")

Cert Module

include cert
Function Description Example
generate(cn, days) Generates a self-signed certificate (mock). cert.generate("localhost", 365)

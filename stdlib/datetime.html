DateTime Module

The datetime module provides tools for working with dates and times.

Importing

include datetime

Current Time

Function Description Example
now() Returns current local timestamp (unix float). let t = datetime.now()
utcnow() Returns current UTC timestamp. let t = datetime.utcnow()
today() Returns timestamp for midnight today. let t = datetime.today()

Formatting & Parsing

Function Description Example
strftime(ts, format) Formats a timestamp as a string. datetime.strftime(t, "%Y-%m-%d")
strptime(str, format) Parses a string into a timestamp. datetime.strptime("2023-01-01", "%Y-%m-%d")
isoformat(ts) Formats timestamp as ISO 8601 string. datetime.isoformat(t)

Calculations

Function Description Example
add_days(ts, n) Adds n days to timestamp. let tomorrow = datetime.add_days(t, 1)
diff_days(ts1, ts2) Returns difference in days. let days = datetime.diff_days(t1, t2)

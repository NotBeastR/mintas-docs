Utilities

A collection of helper modules for CLI, validation, caching, and more.

CLI Module

Tools for building terminal applications, prompts, and progress bars.

include mycli as cli
Function Description Example
prompt(msg, [default]) Prompts user for input. let name = cli.prompt("Name")
confirm(msg) Asks for confirmation (y/n). if cli.confirm("Sure?") { ... }
table(data) Prints a formatted table. cli.table([["ID", "Name"], [1, "Alice"]])
success(msg) Prints a success message in green. cli.success("Installed!")

Colors Module

Terminal text styling and colors.

include colors
Function Description Example
red(text) Returns text wrapped in red ANSI codes. print(colors.red("Error"))
bold(text) Returns bold text. print(colors.bold("Title"))

Validation Module

Input validation for various formats.

include validate
Function Description Example
email(str) Validates email format. validate.email("test@test.com")
url(str) Validates URL. validate.url("https://mintas.dev")
creditcard(str) Validates credit card using Luhn algorithm. validate.creditcard("...")

Cache & Queue

In-memory data management.

include cache
include queue
Function Description Example
cache.set(key, val, [ttl_ms]) Stores a value in cache. cache.set("token", "123", 3600000)
cache.get(key, [default]) Retrieves a value from cache. cache.get("token")
queue.push(q, val) Adds a value to a named queue. queue.push("jobs", {id: 1})
queue.pop(q) Removes and returns front value. let job = queue.pop("jobs")

Algorithms Module

Common computer science algorithms.

include algorithm as algo
Function Description Example
sort(array) Sorts an array (Quicksort). algo.sort([3, 1, 2])
fibonacci(n) Returns the n-th Fibonacci number. algo.fibonacci(10)
levenshtein(s1, s2) Calculates string edit distance. algo.levenshtein("cat", "bat")

Events Module

Event emitter pattern.

include events
Function Description Example
on(event, handler) Registers an event listener. events.on("ready", fn)
emit(event, data) Triggers an event. events.emit("ready", {t: 1})

Fun Utilities

Jokes, ASCII art, and more.

include asjokes as fun

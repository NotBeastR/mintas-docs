Syntax, Types & Operators

Mintas is designed for readability and expressiveness. It supports traditional C-style symbols as well as English-like aliases for a natural coding experience.

Variables & Constants

Variables can be declared using let or its alias so. Constants are declared using const or consta.

let name = "Mintas"   # Standard declaration
so count = 10         # Alias for 'let'
const PI = 3.14       # immutable constant
consta MAX = 100      # Alias for 'const'

English Assignment

You can use the keyword is as an alias for the assignment operator =.

let age is 25
so active is true

Data Types

Mintas is dynamically typed with a rich set of primitives.

Type Description Examples
Number 64-bit floating point numbers. 10, -5.5, 3.14
String UTF-8 text sequences. Supports escapes like \n, \t. "Hello", "Line 1\nLine 2"
Boolean Logical true or false. true, false
Maybe Represents a value that might be present (similar to Option/Result types). maybe
Empty Represents the absence of a value (null/nil). empty

Collections

Arrays (Lists)

Arrays are ordered collections defined with square brackets []. 

let list = [1, 2, 3, "four"]
println(list[0])  # Access elements (0-indexed)

Tables (Dictionaries)

Tables are key-value pairs defined with curly braces {}.

let user = {
    "name": "Alice",
    "age": 30
}
println(user["name"])

Special Features

SuperSet (spr)

The spr keyword wraps an expression in a SuperSet container.

let s = spr { 10 + 20 }

Conditions (cond & follow)

Store conditions for later evaluation.

let check = cond(age > 18)
if follow(check):
    say("Adult")
end

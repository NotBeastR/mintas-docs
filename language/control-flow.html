Control Flow

Mintas offers robust control flow mechanisms with expressive aliases for readability.

Conditionals

The standard if statement requires parentheses around the condition. Code blocks are colon-terminated and must end with end.

Aliases:

  • when can be used instead of if.
  • otherwise can be used instead of else.
let score = 85

if (score > 90):
    say("Excellent")
elif (score > 75):
    say("Good")
otherwise:
    say("Try again")
end

Condensed Logic (cond & follow)

You can encapsulate complex logic into cond objects and evaluate them later using follow. This improves readability for multi-step validation.

let is_adult = cond(age > 18)
let has_ticket = cond(ticket != empty)

if (follow(is_adult) and follow(has_ticket)):
    say("Access Granted")
end

Switch Statement

The switch statement allows for cleaner branching based on a single value.

switch (status):
    case "active":
        say("System is running")
    case "error":
        say("System failure")
    default:
        say("Unknown status")
end

Loops

While Loop

Standard while loop syntax. Parentheses are required.

let i = 0
while (i < 5):
    say(i)
    i++
end

For Loop (Range)

Iterate from a start number to an end number.

for i from 1 to 5:
    say("Count: " + i)
end

For-In Loop (Collections)

Iterate over arrays or tables.

let items = ["apple", "banana", "orange"]
for item in items:
    say(item)
end

Smart Loop (Times)

Use times for a simple counted loop without managing a counter variable manually.

times(5):
    say("Hello!")
end

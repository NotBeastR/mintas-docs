Arrays & Tables

Mintas provides powerful, built-in collection types with a rich set of methods for data manipulation.

Arrays (1-Indexed)

Arrays are ordered lists of values. In Mintas, arrays are 1-indexed, meaning the first element is at position 1.

let fruits = ["Apple", "Banana", "Cherry"]
say(fruits[1]) # Apple

Array Methods

Method Description Example
len() Returns the number of elements. fruits.len()
push(val) Adds an element to the end. fruits.push("Date")
pop() Removes and returns the last element. fruits.pop()
insert(idx, val) Inserts value at position. fruits.insert(2, "Mango")
remove(val) Removes the first occurrence. fruits.remove("Apple")
clear() Removes all elements. fruits.clear()
reverse() Reverses the order in place. fruits.reverse()
contains(val) Checks if value exists. fruits.contains("Banana")
index(val) Returns 1-indexed position. fruits.index("Cherry")
slice(start, end) Returns sub-array. fruits.slice(1, 2)
join(sep) Joins into a string. fruits.join(", ")
sort() Sorts and returns array. fruits.sort()

Functional Methods

let nums = [1, 2, 3, 4, 5]
let doubled = nums.map(lamda(n): n * 2)
let evens = nums.filter(lamda(n): n % 2 == 0)
let sum = nums.reduce(lamda(acc, n): acc + n, 0)

Tables (Key-Value)

Tables (Maps) store data as key-value pairs. Keys are always strings.

let user = { "name": "Alice", "level": 10 }
say($user.name) # Alice

Table Methods

Method Description Example
len() Returns number of pairs. user.len()
keys() Returns array of keys. user.keys()
values() Returns array of values. user.values()
has(key) Returns true if key exists. user.has("name")
remove(key) Removes and returns value. user.remove("level")
merge(table) Merges into this table. user.merge({"role": "admin"})

Web Context Methods

In Dew routes, the getback table has extra methods:

@app.get("/user/>id"):
    let id = getback.param("id")
    let q = getback.query("q")
    let token = getback.header("Auth")
end

Dew Interactive UI & Animations

Dew features a high-performance, reactive front-end runtime and a built-in UI DSL (Domain Specific Language) that allows you to build interactive user interfaces entirely from within Mintas templatesâ€”no external JavaScript required.

1. The <dew> Tag

The <dew> tag is a special element that understands the Dew UI DSL. It allows you to define styles, event handlers, and reactive behavior using a shorthand syntax.

# Inside a .html template
<dew id="my-box" class="card">
    bg: #2d3436
    text-color: "#fff"
    p: 20px
    rounded: 12px
    center: true
    
    text: "Click Me!"
    
    on-click:
        increment
        effect: bounce
    end
</dew>

2. UI Shorthand DSL

Dew maps shorthand properties directly to optimized CSS. Detailed below are the most common properties:

Property CSS Equivalent Example
bg background bg: "#ff0000"
p / m padding / margin p: 10px
px / py padding-x / y px: 2rem
rounded border-radius rounded: 8px
center Flex centering center: true
glass Glassmorphism effect: glass
shadow box-shadow (presets) shadow: lg
w / h width / height w: 100%

3. Reactive State & Binding

Dew templates are reactive. When variables in your Mintas data change, the UI updates automatically.

  • dew-bind="key": Links an element's text/value to a state key.
  • dew-model="key": Two-way binding for inputs.
  • dew-show/hide="key": Toggle visibility based on state.
<input type="text" dew-model="username" />
<h1>Hello, ?( username )?</h1>

4. Built-in Animations

Dew includes a massive library of high-performance CSS animations. You can trigger them from the server using dew.signal() or from the client UI DSL.

The "Super-Power" Animations

Special effects designed to wow users:

  • thanos: Element dissolves into dust and vanishes.
  • glitch: Digital distortion effect.
  • typewriter: Text typing animation.
  • rainbow: Shifting color cycle.
  • neon: Pulsing glowing border.

5. Signals & Commands

Commands allowed inside on-click: or other event blocks:

Command Action
increment [key] Increments a reactive counter.
decrement [key] Decrements a reactive counter.
effect: [name] Triggers a named animation.
toggle [id] Toggles visibility of another element.
show / hide [id] Explicitly sets visibility.
emit [event] Emits a global Dew event.

6. Server Signals

The server can "reach out" and manipulate the browser UI directly using dew.signal().

@app.post("/update"):
    # Change the theme and text of an element remotely
    return dew.signal("alert-box", {
        "theme": "success",
        "text": "Data Saved!",
        "animation": "pop-in"
    })
end

Functions

Functions are reusable blocks of code. Mintas supports multiple keywords for defining functions to suit your coding style.

Defining Functions

You can use func, def, or make to define a function. All are semantically identical.

# Standard Syntax
func greet(name):
    say("Hello, " + name)
end

# Python-style Alias
def add(a, b):
    return a + b
end

# 'make' Alias
make setup_server(port):
    say("Server starting on " + port)
end

Return Values

Use the return keyword to return a value. If no return statement is reached, the function returns empty.

func calculate_area(width, height):
    return width * height
end

let area = calculate_area(10, 5)
say(area) # Prints 50

First-Class Citizens

Functions in Mintas are first-class values. They can be assigned to variables, passed as arguments, or returned from other functions.

let my_func = func(x):
    return x * 2
end

say(my_func(10)) # Prints 20

Classes & OOP

Mintas provides a fully-featured class system with inheritance, encapsulation, and methods.

Defining a Class

Use the class keyword. All members (properties and methods) must be explicitly marked as public or private.

class User:
    public name
    private email

    # Constructor
    public init(name, email):
        this.name = name
        this.email = email
    end

    public getEmail():
        return this.email
    end
end

Instantiation

Use the new keyword to create an instance of a class.

let user = new User("Alice", "alice@example.com")
say(user.name)       # Allowed (public)
say(user.getEmail()) # Allowed (public method)
# say(user.email)    # Error: 'email' is private

Inheritance

Classes can inherit from other classes using the extends keyword.

class Admin extends User:
    public role = "Administrator"

    public init(name, email):
        super.init(name, email) # Call parent constructor
    end
end

Installation

Installing Mintas is straightforward. Depending on your operating system, download the appropriate installer below.

Download Mintas v1.0.0

Windows, macOS, and Linux

Download Installer

Verify Installation

Once installed, open your terminal (Command Prompt or PowerShell on Windows) and type:

mintas --version

You should see something like:

Mintas v1.0.0

Troubleshooting

If the command is not found, ensure that the Mintas bin directory is added to your system's PATH environment variable.

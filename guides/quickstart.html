Quick Start

Welcome to Mintas! This guide will help you write and run your first Mintas program in minutes.

1. Interactive Mode (REPL)

Mintas comes with a Read-Eval-Print Loop (REPL) that allows you to run code instantly. Open your terminal and type mintas without any arguments:

$ mintas
Mintas v1.0.0 (REPL)
> say("Hello from REPL!")
Hello from REPL!
> 10 + 20
30
> exit

2. Your First Script

Create a file named hello.as using your favorite text editor.

# hello.as

# Define a variable
let name = "Developer"

say("Hello, " + name + "!")

# A simple loop using the 'goto' keyword
goto(i = 3 times):
    say("Counting: " + i)
end

3. Running the Script

Run your script from the terminal:

mintas hello.as

You should see:

Hello, Developer!
Counting: 1
Counting: 2
Counting: 3
Did you notice?

Loops in Mintas are 1-indexed by default! The counter i starts at 1, not 0.

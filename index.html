Mintas(Rusty-Mintas)

A modern, high-performance programming language designed for certainty and speed.

1-Indexed

Arrays and lists start at 1, making them more intuitive for mathematical operations and human logic.

Three-Boolean Logic

Beyond true and false, Mintas introduces maybe, handling uncertainty explicitly.

Fast Performance

Compiles using the JetX engine for near-native execution speeds.

Readable Syntax

Designed to be read like English. Clean, expressive, and devoid of unnecessary boilerplate.

Dynamic but Strict

Enjoy the flexibility of dynamic typing with the safety of strict runtime checks.

Catch Errors Early

Advanced static analysis catches potential errors before execution begins.

Strong Uncertainty

Robust handling of nulls, unknown states, and errors.

Hello World

Your first Mintas program is as simple as it gets. Create a file ending in .as:

# hello.as
say("Hello World")

Run it with: mintas hello.as
